## Construction Rules

- Populate target packets independently before comparing them.
- Populate null packets with the same field and gate burden as target packets.
- Use primary technical or scientific sources for material claims.
- Do not use frozen WI-069 exploration artifacts as evidence; they are origin
  history only.
- Keep unlike quantities unlike unless a typed conversion map is explicitly
  supplied.
- Record imports and losses before agency, escape, or feedback scoring.
- Leave a field open when the required evidence has not been supplied.

## Stage Order

1. `scaffolded`: file contains the full field and gate shape.
2. `populated`: material fields cite source evidence, with open fields marked.
3. `frozen`: a packet snapshot is ready for the CL-001 contrast matrix.
4. `scored`: gate receipts have been assigned with evidence references.

The CL-001 verdict can only be written after all four packets have reached the
same scoring standard.
