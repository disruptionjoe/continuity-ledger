## Purpose

This file indexes the first bounded experiment for the proposed emergence/collapse/class-relative escape research program.

The experiment asks:

> Can a small set of accounting objects classify divergent examples of collapse and emergence without becoming metaphor, duplicating existing fields, or requiring an assumed universal invariant?

## Objects Under Test

1. `ObstructionTransitionPacket`
2. `BoundaryImportPacket`
3. Productive / terminal / captured closure typology
4. Completion-factorization test
5. No-go / escape / residue ledger

## Gate

The experiment supports a new repo only if the accounting objects produce discriminating, non-forced structure across divergent examples.

The experiment fails or narrows if:

- packet fields are mostly empty or metaphorical;
- every example is better handled entirely inside an existing discipline;
- the closure typology cannot separate productive closure from collapse;
- boundary imports are so universal that they stop discriminating;
- no-go/escape accounting produces documentation but no formal or executable pressure.
